Documentation Page

Overview

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Get a list of a user's favorite movies /users/:Username/favorites GET None A JSON object holding data about a user's favorite movies.
Allow users to add a movie to their favorites /users/:Username/favorites/:MovieID' POST None A text message indicating the movie has been added to the user's favorites.

Allow existing users to delete their account /users/:Username DELETE None A text message indicating the user has been deregistered.
